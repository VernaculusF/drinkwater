# 📱 Инструкция по подключению Яндекс РСЯ

## 1. Регистрация и получение Block ID

### Шаг 1: Регистрация в Яндекс Рекламной Сети (РСЯ)
1. Перейдите на сайт: https://partner.yandex.ru/
2. Войдите с помощью аккаунта Яндекс (или зарегистрируйтесь)
3. Выберите "Мобильные приложения" → "Добавить приложение"

### Шаг 2: Добавление приложения
1. В разделе **"Приложения"** нажмите **"Добавить приложение"**
2. Выберите платформу: **Android**
3. Укажите:
   - **Название приложения**: DrinkWater (или ваше)
   - **Пакет приложения**: `com.example.drinkwater` (из `android/app/build.gradle.kts`)
   - **Store URL**: пока пусто (заполните после публикации в Google Play)

### Шаг 3: Создание рекламного блока (Block)
1. После добавления приложения перейдите в **"Рекламные блоки"**
2. Нажмите **"Создать блок"** → **"Баннер"**
3. Выберите:
   - **Тип**: Sticky Banner (фиксированный внизу)
   - **Размер**: 320x50 или Adaptive (адаптивный)
4. Нажмите **"Создать"**

### Шаг 4: Получение Block ID
После создания блока вы получите **Block ID** (например: `R-M-XXXXXX-Y`)

**ВАЖНО**: Скопируйте этот ID — он понадобится для настройки приложения!

## 2. Настройка проекта Flutter

### Шаг 1: Установите зависимость (УЖЕ СДЕЛАНО)
```bash
flutter pub get
```

Зависимость `yandex_mobileads: ^7.5.0` уже добавлена в `pubspec.yaml`.

### Шаг 2: Замените тестовый Block ID на ваш реальный

Откройте файл `lib/constants/app_constants.dart` и замените:

```dart
// БЫЛО (тестовый ID):
static const String yandexAdBannerUnitId = 'demo-banner-yandex';

// СТАНЕТ (ваш реальный ID):
static const String yandexAdBannerUnitId = 'R-M-XXXXXX-Y'; // Замените на свой!
```

### Шаг 3: Настройте AndroidManifest.xml

Откройте `android/app/src/main/AndroidManifest.xml` и добавьте **внутри тега `<application>`**:

```xml
<application>
    <!-- Существующий код приложения -->
    
    <!-- Meta-data для Яндекс РСЯ -->
    <meta-data
        android:name="com.google.android.gms.ads.APPLICATION_ID"
        android:value="ca-app-pub-3940256099942544~3347511713"/>
</application>
```

**Примечание**: Этот ID - тестовый от Google AdMob (может потребоваться для совместимости SDK).

### Шаг 4: Добавьте разрешения в AndroidManifest.xml

В начале файла (до `<application>`):

```xml
<manifest>
    <!-- Существующие разрешения -->
    
    <!-- Разрешения для рекламы -->
    <uses-permission android:name="android.permission.INTERNET"/>
    <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE"/>
</manifest>
```

(Эти разрешения скорее всего уже есть, проверьте)

## 3. Тестирование

### Тестовый режим (СЕЙЧАС АКТИВЕН)
Пока в коде указан `demo-banner-yandex` — используется **тестовый баннер**.

**Преимущества**:
- ✅ Не нужен реальный Block ID

- ✅ Можно тестировать без регистрации

```bash
flutter run
```

На главном экране внизу должен появиться баннер.

**Если баннер не загружается**:
1. Проверьте логи: `flutter logs` (ищите сообщения с `⚠️` или `Yandex`)
2. Убедитесь, что устройство подключено к интернету
3. Проверьте, что инициализация прошла успешно (в логах должно быть `✅ Яндекс РСЯ SDK инициализирован`)

## 4. Переход на продакшн (реальную рекламу)

### Шаг 1: Замените Block ID
В `lib/constants/app_constants.dart`:

### Шаг 2: Соберите Release APK
```bash
flutter build apk --release
```

### Шаг 3: Опубликуйте в Google Play
1. Загрузите APK в Google Play Console
2. Заполните описание, скриншоты
3. Опубликуйте приложение

Вернитесь в кабинет Яндекс РСЯ:
1. Откройте настройки вашего приложения
2. Укажите **Store URL** (ссылку на приложение в Google Play)
3. Сохраните изменения

### Шаг 5: Дождитесь модерации
Яндекс проверит ваше приложение (обычно 1-3 дня). После одобрения начнет показываться реальная реклама.

## 5. Отслеживание доходов

В личном кабинете Яндекс РСЯ доступны:
- **Показы** (impressions) — сколько раз показался баннер
- **Клики** (clicks) — сколько раз пользователи нажали на рекламу
- **CTR** (Click-Through Rate) — процент кликов
- **eCPM** (effective Cost Per Mille) — заработок за 1000 показов
- **Доход** — сколько вы заработали

**Выплаты**: Яндекс выплачивает доход при достижении минимального порога (обычно 500-1000 рублей).

## 6. Часто задаваемые вопросы (FAQ)

### Где найти API для подключения?
API встроен в SDK `yandex_mobileads`. Вы используете его через:
- `MobileAds.initialize()` — инициализация
- `BannerAd(adUnitId: ...)` — создание баннера
- `bannerAd.load()` — загрузка рекламы
- `AdWidget(bannerAd: ...)` — виджет для отображения

Весь код уже реализован в `lib/services/ad_service.dart` и `lib/screens/home_screen.dart`.

### Почему баннер не показывается?
Возможные причины:
1. **Нет интернета** — проверьте подключение
2. **Неверный Block ID** — убедитесь, что скопировали правильно
3. **Приложение не одобрено Яндексом** — дождитесь модерации
4. **Низкий fill rate** — Яндекс не всегда имеет рекламу для показа

### Сколько можно заработать?
Зависит от:
- Количества активных пользователей
- Географии (в России eCPM выше)
- CTR (насколько пользователи кликают на рекламу)

**Примерная оценка**: 1000 активных пользователей могут приносить 1000-5000 руб/месяц.

### Можно ли добавить Rewarded Video (видео с вознаграждением)?
Да! Это следующий этап (в TODO). Rewarded Video приносит больше дохода, так как пользователи сами выбирают просмотр.

### Нужно ли платить за использование Яндекс РСЯ?
Нет, Яндекс РСЯ бесплатен для разработчиков. Яндекс зарабатывает на рекламодателях, а вам платит процент от их расходов.
